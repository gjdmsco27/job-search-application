# 📱 Job Search App (React Native)

Rapid API를 활용하여 실시간 구직 정보를 검색하고 조회할 수 있는 React Native 애플리케이션입니다.
Expo Router를 기반으로 파일 시스템 라우팅을 구현하였으며, 국내 사용자를 위해 **전면 한글화(Localization)** 및 **UI/UX 최적화** 작업을 수행하였습니다.

## ✨ 주요 기능 (Key Features)

### 1. 기본 기능 (Core Features)
- **홈 화면 (Home):**
  - **개인화된 환영 메시지:** "안녕하세요 허은채님 👋" 및 상황에 맞는 인사말 제공
  - **직무 검색:** 키워드 입력 및 검색 기능 (한글 Placeholder 적용)
- **채용 공고 리스트:**
  - **Popular Jobs:** 가로 스크롤 가능한 인기 채용 리스트
  - **Nearby Jobs:** 세로 리스트 형태의 주변 채용 정보
- **상세 페이지 (Job Details):**
  - 채용 공고의 상세 내용 탭(소개, 자격요건, 업무내용) 구현 및 데이터 연동
  - 회사 정보, 로고 표시 및 구글 채용 페이지 링크 연결

### 2. 추가 구현 기능 (Custom Features) 🚀
기본 과제 외에 추가로 직접 기획하고 구현한 기능들입니다:

- **🍔 카테고리 메뉴 (Category Menu):**
  - **채용 형태 필터:** 메인 화면 대신 전용 메뉴 페이지에서 `정규직`, `파트타임`, `계약직` 공고를 선택하도록 구조 개선
  - **한글-영어 매핑:** 화면에는 한글로 표시되지만, API 요청 시에는 영어 코드(`Full-time` 등)로 변환되어 검색되도록 로직 구현
  - **앱 정보 하단 고정:** 앱 버전(v1.0.0) 및 개발자 정보(Developed by 허은채) 표시

- **👤 마이 페이지 (My Profile):**
  - 메인 화면 우측 상단 프로필 버튼을 통해 접근 가능
  - 사용자 프로필 이미지(커스텀 스타일링) 및 정보 표시 화면 구현

- **👀 더보기 연결 (Show All):**
  - **Popular / Nearby 섹션 연동:** 각 섹션의 '모두 보기(Show all)' 버튼 클릭 시, 해당 카테고리의 **전체 검색 결과 페이지**로 이동하도록 라우팅 연결
  - **헤더 타이틀 매핑:** 검색 결과 페이지 진입 시 `Popular` → "인기 채용 공고", `Nearby` → "주변 채용 공고"로 자동 변환하여 표시

- **📤 공유하기 (Share):**
  - 채용 상세 페이지 우측 상단 공유 버튼 활성화
  - `Share` API를 사용하여 카카오톡, 메시지 등 외부 앱으로 공고 링크 공유 가능

### 3. 🎨 UI/UX 및 한글화 (Localization & UX)
사용자 편의성을 위해 디자인과 가독성을 대폭 개선하였습니다:

- **🇰🇷 전면 한글화 (Localization):**
  - **헤더 & 메뉴:** 불필요한 영문 타이틀(`Header`) 제거 및 모든 메뉴명 한글화
  - **검색 결과:** 검색된 키워드(Full-time 등)를 자동으로 한글(정규직 등)로 변환하여 표시하고, "Job Opportunities"를 "채용 공고"로 변경
  - **상세 페이지:** 탭 이름을 `소개`, `자격요건`, `업무내용`으로 변경하고 섹션 제목 한글화

- **📖 가독성 및 레이아웃 최적화:**
  - **상세 페이지 개선:** 자격요건/업무내용 리스트의 불렛 포인트(점)와 텍스트가 어긋나지 않도록 `flex-row` 레이아웃 적용
  - **스타일 조정:** 줄 간격(Line Height)을 넓히고, 가독성이 좋은 짙은 회색(`#444262`) 컬러를 적용하여 정보 전달력 강화
  - **디자인 포인트:** 메인 환영 메시지에 포인트 컬러(`#2563EB`)와 이모지를 적용하여 전문적인 느낌 강조

- **🛡️ 데이터 안정성 확보 (Mock Data):**
  - API에서 상세 데이터(Description 등)가 비어있을 경우, 빈 화면 대신 "상세 정보가 없습니다" 등의 기본 문구가 표시되도록 예외 처리 구현

## 🛠️ 기술 스택 (Tech Stack)
- **Framework:** React Native / Expo
- **Routing:** Expo Router (v2)
- **Language:** JavaScript
- **API:** Rapid API (JSearch)
- **Styling:** React Native StyleSheet
